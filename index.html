Título. Descripción de la SdA

  • Etapa educativa y nivel al que se dirige: esta SdA está dirigida a alumnado de 1º ESO
  • Áreas y/o materias  implicados: Biología y Geología
  • Agrupamientos y organización espacial: 
    •  Grupos heterogéneos. Este agrupamiento se utiliza en tareas cooperativas se trata de agrupamientos de parejas, tres o cuatro alumnos con distintas características, favorece la participación y colaboración en un objetivo común.
    •  Gran grupo o grupo-clase. Grupo aula completo, este agrupamiento se utiliza para presentar información al grupo, presentar un grupo de actividades o hacer una puesta en común de las actividades individuales o grupales. 
    •  Equipos flexibles. Se trata de un agrupamiento homogéneo de alumnos según las capacidades y necesidades del alumnado para atender los diferentes ritmos de aprendizaje, la agrupación se hace a partir de un grupo heterogéneo.
    • Trabajo individual. Se utiliza para que el alumno interiorice lo aprendido o en las actividades de consolidación
  • Temporalización: 48 sesiones
  •  Descripción de los productos: Los productos finales desarrollados por el alumnado son: cuaderno clase, cuaderno laboratorio, fichero digital, maquetas, recortables, producciones escritas, una infografía y un breakout. 
  • Herramientas tecnológicas y otros recursos complementarios: Plataforma Teams, software educativo de Anaya, canales de youtube (Bio[ESO]sfera, biologiammb), canales de divulgación científica (El Robot de Platón, C de Ciencia, La Hiperactina) Kahoot!, Quizizz, Genially, Canva, Microsoft Forms...
